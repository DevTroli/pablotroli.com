> Hub pessoal em `pablotroli.com` — ponto central da minha presença online.

## Objetivo

Um hub estático que centraliza quem eu sou na internet:
blog, projetos do GitHub, redes sociais — tudo num lugar só
com uma identidade visual forte (terminal/hacker aesthetic).

## Decisões tomadas

| Decisão    | Escolha                             | Motivo                                                      |
| ---------- | ----------------------------------- | ----------------------------------------------------------- |
| Stack      | HTML + CSS + JS puro                | Isolamento total do blog, zero dependências, deploy simples |
| Hospedagem | GitHub Pages                        | Simples como meu projeto                                    |
| Domínio    | pablotroli.com via Cloudflare       | Já configurado                                              |
| Blog       | blog.pablotroli.com (Hugo + Hextra) | Stack separada, link externo                                |
| Dark mode  | Não se aplica                       | Identidade terminal é sempre dark                           |

## Stack

- **HTML/CSS/JS** — sem frameworks
- **Fonts:** JetBrains Mono + Share Tech Mono (Google Fonts)
- **Deploy:** Github Pages
